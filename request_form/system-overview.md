# 정신병원입원 의뢰 시스템 - 역할별 화면 및 기능 명세

## 📋 시스템 개요

정신과 응급환자 발생 시 경찰관이 적절한 병원을 찾고 입원 의뢰서를 작성하여 전송하고, 병원에서 이를 검토하여 승인/거부 처리하는 시스템입니다.

---

| 분류 | 화면 명 |
